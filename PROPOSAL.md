## What are you building?

I am building an internship tracker web app designed for students and early-career professionals who are currently in internships. It helps users log their daily work, store files and notes, track networking interactions, and automatically generate polished weekly updates to send to their manager.

## Why?

During internships, it’s often difficult to remember everything you’ve worked on, summarize your progress clearly, and maintain professional communication with your manager. This app solves that by centralizing work logs, files, and reflections into one place, and turning them into structured weekly updates. It also helps users track networking conversations and relationships, which are easy to forget but important for long-term career growth.

## MVP vs. Stretch Goals:

# MVP (Minimum Viable Product):

Dashboard to log daily/weekly journal entries of work completed
File upload system (PDFs, images, etc.) tied to weekly logs
Weekly summary generator based on user inputs
Editable template for manager updates
Preview box for generated weekly message (copy/paste ready)
Simple networking tracker (name, email, notes, last meeting date)

# Stretch Goals:

Tone adjustment buttons (formal, casual, concise, etc.) for summaries
Smart summarization using AI (auto-generate summaries from logs)
Reminder system for sending weekly updates
Networking follow-up reminders (e.g., “reach out again in 2 weeks”)
Photo/memory gallery for documenting internship experiences
Search and filter functionality across logs and contacts
Analytics view (e.g., what skills/tasks you’ve focused on over time)

## What technologies do you plan to use?
Frontend: HTML, CSS, JavaScript
Storage: LocalStorage or Firebase (for saving logs, files, and contacts)
File Handling: Browser file upload APIs
AI Integration (optional/stretch): OpenAI API for summary generation and tone adjustment
Email/Copy Workflow: Simple UI for copying generated summaries (potential stretch: email integration)
UI Libraries (optional): Tailwind CSS or a component library for faster design
